- การติดตั้ง 

นำโฟลเดอร์ tmwqr ไปใส่ในเว็บของของคุณ
เข้าหน้าเว็บด้วย url https://[yourdomain]/tmwqr จะมีหน้าติดตั้งขึ้นมา ให้กรอกข้อมูลตามหน้าติดตั้ง เมื่อเสร็จขั้นตอนติดตั้งแล้วจะมีไฟล์  config.php ขึ้นมา สามารถแก้ไขค่าต่างๆภายหลังได้ที่ไฟล์นี้

URL API web hook สำหรับนำไปใส่ที่แอปทรูวอเลทคือ https://[yourdomain]/tmwqr/hook_action.php


- สำหรับนักพัฒนาเพิ่มเติม ต้องมีพื้นฐานการใช้ภาษา php sql html css หรือติดต่อ Admin ในการขอใช้บริการทำโค้ดเพิ่ม

การแก้ไขโค้ดเมื่อเติมเงินสำเร็จ ให้ทำการอัพเดทฐานข้อมูลต่างๆเพิ่มเติม อยู่ที่ไฟล์ hook_action.php
หน้าตาดีไชน์ เว็บเติมเงิน แก้ไขได้ที่ไฟล์ index.php
